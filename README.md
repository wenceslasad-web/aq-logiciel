# Projet d'assurance qualité logicielle

Ce projet a été créé dans le cadre d'un laboratoire sur GitHub afin de pratiquer la gestion d'un dépôt, l'utilisation d'une licence MIT, la création d'une issue, la rédaction d'un fichier Markdown et la mise en ligne d'un site web avec GitHub Pages. Il contient une page web simple avec mon nom, un fichier de licence, un fichier README et un site publié en ligne.


Site web : https://wenceslasad-web.github.io/aq-logiciel/
